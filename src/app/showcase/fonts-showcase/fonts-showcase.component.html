The font sizes can be set by either using the font classes, e.g. text--large, or by using the right tags like H1, H2, H3 etc.

Font can also be set with the font-size function directly on your own classes, e.g:

  .my-own-class {{'{'}}
    font-size: font-size("xxl");
    ...
  {{'}'}}
Display
font-size: font-size("display); font-weight: font-weight("normal"); CLASS: text--display
