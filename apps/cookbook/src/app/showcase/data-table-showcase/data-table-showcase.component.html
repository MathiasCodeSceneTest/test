A data table displays similar information in rows and columns.

The Kirby Data Table adds styling on top of the built in table HTML element which makes it very flexible as consumers are in complete control of the layout.

Please note: This also means that Kirby Data Table is not intended for highly complex use cases where large datasets need to be manipulated and rendered efficiently.
